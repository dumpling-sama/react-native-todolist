## Sample Todo List App implemented in React Native 

### To Run:
1. Open Terminal
2. Type 'npm install' or 'yarn install'
3. Type 'cd ios && pod install' to install dependencies for iOS 
4. Type 'react-native run-ios' or 'react-native run-android'

### Features:
1. Add Todo
2. Update Todo
3. Delete Todo
4. View All Todos
